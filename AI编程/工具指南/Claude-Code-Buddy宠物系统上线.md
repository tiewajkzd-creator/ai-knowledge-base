## 背景

今天Claude Code悄悄上线了Buddy宠物模式，输入/buddy即可孵化专属宠物。4月1日首次亮相，恰逢后天复活节。

## Buddy系统设计

### 双层架构：Bones + Soul
- **骨架层（Bones）**：决定宠物外观
  - 用userID + 固定值，通过Mulberry32随机生成
  - 确定物种、眼睛、帽子、稀有度等外观属性
  - 绑定userID，换电脑/重装Claude Code宠物不变
- **灵魂层（Soul）**：决定宠物性格
  - 存在本地，可以重新roll
  - 第一次孵化时，Claude根据物种和属性给它取名、写性格描述

### 宠物属性（5大属性）
- 调试能力、耐心值、混乱值、智慧值、毒舌值
- 稀有度越高，属性基础值越高
- 每只宠物有一个突出属性 + 一个短板属性

### 稀有度机制
- 6个稀有度等级（普通→传说）
- 闪光（Shiny）独立1%概率，与稀有度无关
- 18种宠物：鸭子、蘑菇、猫头鹰、蜗牛、幽灵、水豚（卡皮巴拉）等

## 重Roll漏洞（传说闪光卡皮巴拉）

### 原理
Claude Code正常登录时，accountUuid绑定账户无法伪造。但通过环境变量`CLAUDE_CODE_OAUTH_TOKEN`登录时，不会把accountUuid写入~/.claude.json，buddy命令会退而求其次读取userID字段——而这个字段可以随便改。

### 操作步骤
1. 用`claude setup-token`拿到OAuth token
2. 删掉~/.claude.json，清掉旧的accountUuid
3. 写入最精简配置（只留hasCompletedOnboarding和theme字段）
4. 用环境变量方式启动Claude Code
5. 运行buddy-reroll.js脚本，暴力枚举（5000万次）找到想要的宠物seed
6. 把目标userID塞进~/.claude.json，重新打开Claude Code，输入/buddy

### 彩蛋
- 闪光传说卡皮巴拉是强度党的胜利
- 刷初始SSR的快感，在命令行里重现了
