## 摘要

作者分享了一个自己用了两年、用于快速深度研究的 Prompt 框架——**「横纵分析法」**。结合 AI 可在半小时内完成一份一两万字的陌生领域研究报告。

方法论脱胎于社会科学和语言学的经典研究视角（索绪尔的历时/共时分析），已开源在 GitHub。

## 核心方法论：横纵分析法

### 纵向分析（Diachronic / Longitudinal）
沿时间轴，完整还原研究对象从诞生到现在的发展全貌。

**包含维度：**
- 起源追溯：诞生背景、技术/理念/需求来源
- 诞生节点：首次发布时间、最初形态和定位
- 演进历程：关键节点、融资事件、团队变动、战略转型
- 决策逻辑：每个节点背后的选择与约束条件

### 横向分析（Synchronic / Cross-sectional）
以当前时间点为切面，与同赛道竞品进行全面对比。

**对比维度：**
- 核心差异：技术路线、产品形态、目标用户、定价策略
- 用户视角：真实口碑、社区评价、优缺点
- 生态位分析：在整个赛道版图中的位置
