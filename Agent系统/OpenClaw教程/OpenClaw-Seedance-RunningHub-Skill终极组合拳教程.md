## 核心观点

1. **OpenClaw + RunningHub Skill 实现"自然语言调用一切AI能力"：** 通过安装 RHClaw 技能（GitHub开源），OpenClaw 可以直接调用 RunningHub 平台的 179 个标准 API 端点，覆盖文生图、文生视频、音频、3D 模型等全品类生成能力，以及无限自定义 ComfyUI 工作流应用。

2. **Seedance 2.0 在线工作流无需本地部署：** 通过 RunningHub 云端运行 Seedance 2.0 工作流，无需本地 GPU 和环境配置，降低了 0 基础用户的使用门槛。配合 OpenClaw 的 Skill 机制，用自然语言即可触发视频生成。

3. **视频生成模型覆盖全面，支持智能选择：** RHClaw 集成了 Google Veo 3.1、Grok Video、Kling v3.0 Pro、Sora、MiniMax Hailuo、Seedance v1.5 Pro 等 8 款主流视频模型，用户可手动选择或由 Agent 自动匹配最佳模型。

4. **图片生成同样丰富：** 包含全能图片PRO（nanobanana Pro 同款）、全能图片V2（nanobanana 2 同款）、悠船 v7（Midjourney 风格）、悠船 niji7（二次元风格）、Seedream v5（字节写实风格）共 5 款精选模型。

5. **飞书机器人部署教学：** 本期教程还包含将 OpenClaw Agent 部署为飞书机器人的完整流程，实现在飞书中直接与 AI Agent 对话、触发生成任务。

## 实用工具/技巧

### RHClaw Skill 安装方式
在 OpenClaw 对话中直接发送：
> 从 https://github.com/HM-RunningHub/OpenClaw_RH_Skills 安装 RunningHub 技能

Agent 会自动克隆仓库、复制文件、引导配置 API Key。
