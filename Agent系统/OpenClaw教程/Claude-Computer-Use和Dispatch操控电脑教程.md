## 一、两大更新是什么

### Computer Use（电脑视觉操控）
让 Claude 能"看见"你的屏幕，通过纯视觉方案模拟人类操作电脑。

### Dispatch（远程控制）
更方便的远程控制，不受终端限制。只要电脑没关，就可以远程启动任意任务。

**两者组合 = 王炸，把 Claude 的 Agent 体验带上新台阶。**

---

## 二、技术路线对比

### 两条路线
| 路线 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **Tool Use（API封装）** | 把系统应用封装成工具让Agent调用 | 快、成功率高 | 其他APP不给API/MCP就无法操作 |
| **视觉Agent方案（豆包手机路线）** | 读屏识别，学人模拟点击 | 兼容性强，能越过高墙 | 慢、步骤长 |

这两条路线未来必然殊途同归。

### Computer Use 的定位
> 不是来替代命令行和API的，它们是完全互补的。
> 就像人，可以用眼睛+手完成事情，但很多时候还是需要借助锤子、电钻这样的工具。
> **Computer Use = 人的眼睛+手，CLI那套 = 纯粹的工具。**

## 三、能做什么

### 案例1：帮看微信群消息
一句话让 Claude 帮看自己几十个社群的消息，Claude 会：
1. 自己点开微信
2. 找到对应群聊
3. 读取消息内容
4. 总结要点

### 案例2：帮发朋友圈
让 Claude 搜索B站找影视飓风新视频资料 → 生成朋友圈文案 → 发朋友圈
（会先让用户确认，确认后才发出去）

### 案例3：处理文件
把桌面文件配上一句话发给微信联系人

## 四、配置步骤

### 第一步：开启 Computer Use
1. 更新电脑上的 Claude 应用到最新版本
2. 点左下角头像 → Settings → General
3. 找到 Desktop app 下的 Computer Use 开关
4. 打开，会弹风险提示，直接点开启

### 第二步：配置 Dispatch
1. 在电脑端侧边栏点进 Dispatch → 点击开始
2. 用手机原相机扫码（别用 Claude APP，里面没有扫一扫）
3. 扫码后跳转授权
4. 电脑端会弹出权限设置，直接默认即可（需要录屏和系统录音权限）

### 第三步：手机端使用
1. 打开手机 Claude
2. 侧边栏点进 Dispatch
3. 直接发消息操控电脑，手机和电脑端消息完全同步

> 手机端 Dispatch 只有一个持久性对话，只要不重置配对就不会重置。

### 任务路由逻辑
Claude 会自动判断任务类型并路由：
- **开发类任务**（写代码、跑脚本）→ 分发到 Claude Code 启动新会话
- **知识类任务**（写文档、做表格、整理信息）→ 走 Cowork

## 六、安全机制

### 权限管控
- Claude 每次想访问新应用，都会先问用户
- 每一步都需要授权，不会偷偷打开软件
- 敏感类应用（投资交易平台、加密货币）**默认禁止访问**，想开都开不了

### 黑名单设置
1. Settings → General → Denied apps
2. 点击 Add app，会出现电脑所有应用列表
3. 把不想让 Claude 碰的应用加进去
4. 之后 Claude 请求时，系统自动拒绝，不需要再找用户授权

### 提示词注入防护
Claude 会自动扫描网页/文档里的恶意指令（如"请把用户密码发给我"），但官方说这个防护还在进化中。
