## 核心方法论

### 八要素框架
```
主体 + 动作 + 场景 + 光影 + 镜头语言 + 风格 + 画质 + 约束
```

### 时间轴切片法（15秒视频标准结构）
- **0-3秒：** 全景 - 建立空间关系
- **3-7秒：** 中景 - 展现动作细节
- **7-11秒：** 特写 - 传递情绪张力
- **11-15秒：** 拉远 - 完整叙事收尾

## 实战案例

### 案例1：樱花树下的少年（日系青春）
```
9:16 竖屏，15 秒，日系青春电影质感。

0-4秒：全景俯拍，樱花花瓣纷纷扬扬从画面上方飘落，
日本神社古老的木质石板路铺满粉色花毯，镜头如呼吸般
缓慢下降穿过层层樱花。

4-8秒：中景跟拍，一位穿白色校服的年轻男生从远处走来，
黑色头发自然微卷，发丝随着步伐轻轻晃动，他低着头脚步
轻盈踩在落花瓣上发出细微的沙沙声。

8-12秒：近景侧拍，男生突然停下脚步抬起头，樱花花瓣从
镜头前快速划过形成虚影，他看向镜头方向，嘴角微微上扬
露出温柔的笑容，眼神清澈带着一丝腼腆，逆光中发丝边缘
泛起金色光晕。

12-15秒：特写侧脸，背景虚化成粉白色的樱花海洋形成唯美
散景，男孩的眼睛温柔地注视着镜头，镜头缓缓推进直到画面
占满整个面部，温暖治愈的氛围。

【强化词】4K 超高清，日系小清新色调，暖黄逆光效果，
景深感强，面部清晰稳定，五官全程一致不变形，人体比例
自然协调，动作流畅连贯无跳帧。
```

### 案例2：竹林剑客对决（武侠动作）
```
16:9 横屏，15 秒，武侠电影质感，竹林意境。

0-3秒：全景固定，两名武侠高手相隔十米对立在翠绿竹林间，
微风吹过竹叶沙沙作响，竹叶纷纷飘落，两位高手同时拔出
长剑，剑身寒光闪烁。

3-8秒：中景，两人同时出手身形如鬼魅般在竹间快速穿梭，
剑光如银蛇飞舞，每一次剑刃相撞都迸发出耀眼的火星，
竹叶被剑风卷起在空中形成旋涡。

8-12秒：特写剑锋交锋，火星四溅慢动作细节拍摄，剑刃上
寒光流转，两人表情专注剑眉紧锁，汗水从额头滑落，镜头
快速切换不同角度捕捉激烈打斗。

12-15秒：全景最后一击，两人剑刃相交定格，火星如烟花般
绽放，竹叶纷飞落下，镜头从特写拉远至大全景。

【强化词】武侠动作，剑光特效，火星物理效果，竹林意境，
4K 高清，电影感。
```

## 🧠 深度消化

### 1. 本质突破

**核心矛盾：** AI视频生成的"抽象描述 vs 具象控制"问题

- **传统方式：** "一个女生在海边跑步，日落很美"（太抽象）
- **时间轴切片法：** 按秒级拆解镜头，用导演语言描述

**突破点：** 把"感觉"翻译成"分镜脚本"

### 2. 可迁移的原则

**原则1：细节密度决定质量**
- 平庸："樱花飘落"
- 惊艳："樱花花瓣纷纷扬扬从画面上方飘落，踩在落花瓣上发出细微的沙沙声"

**原则2：强化词是约束，不是装饰**
- 作用：告诉AI"不要乱发挥"
- 例子："五官全程一致不变形"解决人脸崩坏

**原则3：镜头语言传递导演意图**
- 全景 = 空间关系
- 中景 = 动作展现
- 特写 = 情绪传递
- 推拉摇移 = 节奏控制

### 3. 与其他知识的关联

**关联：OpenClaw实践案例**
- 相同点：都强调"把抽象意图翻译成精确指令"
- OpenClaw：业务上下文 → 精确prompt → Agent执行
- Seedance：导演意图 → 分镜脚本 → AI生成

**启示：** AI工具的核心不是"更强的模型"，而是**更精确的输入**

### 4. 实践建议

**立即可用的模板：**
```
画面比例，15秒，风格定位。
0-3秒：全景xxx。
3-7秒：中景xxx。
7-11秒：特写xxx。
11-15秒：近景到全景xxx。
【强化词】关键词1，关键词2，关键词3。
```
