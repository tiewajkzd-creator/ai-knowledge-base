## 📋 Skill体系总览（5大核心Skill）

### 1. Dialogue_Scene_Designer — 对话场景镜头序列
**场景**：两人/多人对话

**核心能力**：
- 正反打镜头（Over-the-Shoulder）
- 过肩镜头双人序列
- 180°规则保持空间连贯
- 反应镜头链（Reaction Shot Chain）
- 视线匹配（Eyeline Match）
- 三角构图（Triangular Composition）

**配套工具**：
- 对话节奏引擎：对话密度与镜头节奏匹配
- 视线修正系统：保证正反打逻辑一致
- 视线匹配表：解决跨轴问题

**基础模板**：标准5镜头对话序列、紧张对话序列、亲密对话序列

---

### 2. Action_Sequence_Designer — 动作戏镜头序列（打斗/追逐/爆破）

**三大核心系统**：

**① Fight_Sequence（打斗镜头）**
- 基础攻击序列：Setup → Attack_Prep → Attack_Execution → Impact → Reaction → Recovery
- 连招序列：三连击、反反击、格挡反击
- 镜头类型：Master_Wide / Action_Medium / Impact_Close / POV_Shots
- 强度等级：low（1-2s/镜头）/ medium（0.5-1s）/ high（0.3-0.5s）

**② Chase_Sequence（追逐镜头）**
- 追逐镜头语法：保持方向一致性、营造速度感
- 速度技巧：Motion_Blur / Rapid_Cutting / Dutch_Angle / Foreground_Elements
- 序列模板：foot_chase / car_chase

**③ Explosion_Sequence（爆破镜头）**
- 时间轴设计：pre_explosion → explosion → post_explosion
- 镜头序列：Setup(2-5s) → Trigger(0.5-1s) → Initial_Blast(0.2-0.5s) → Fireball(0.5-1s) → Debris(1-2s) → Aftermath
- 相机技巧：轻微震动、快速推拉、多角度快速剪辑

**节奏控制引擎（Rhythm_Engine）**
- 根据动作强度自动调整剪辑速度
- intensity_mapping: calm / building / peak / release
- breathing_points：每15-20秒需要停顿、大动作后恢复

**空间定位系统（Spatial_System）**
- orientation_markers：固定参考物、地面标记、背景特征
- coverage_strategy：定期全景重定位、匹配剪辑保持方向、避免180°翻转

---

### 3. Emotional_Sequence_Designer — 情感戏镜头序列（爱情/悲伤/冲突）

**三大核心系统**：

**① Romance_Sequence（爱情镜头）**
- 亲密感营造三阶段：Attraction → Connection → Intimacy
- 关键镜头：Glance_Shot / Eye_Contact / First_Touch / Kiss_Sequence
- 构图技巧：浅景深、柔和光线（金色时刻）、框架构图、反射元素

**② Grief_Sequence（悲伤镜头）**
- 悲伤四阶段：Shock → Processing → Release → Recovery
- 视觉隐喻：下雨/落叶/空椅子/褪色照片/关闭的门
- 核心原则：慢节奏、长镜头停留、环境音保留

**③ Conflict_Sequence（冲突镜头）**
- 张力积累三阶段：Setup → Escalation → Climax
- 冲突技巧：Dutch angle / 推轨变焦（Vertigo效果）/ 色彩对比 / 空间压缩
- 节奏模式：1-2-3逐渐加速

**情绪节奏引擎（Emotional_Rhythm）**
- 情绪弧线匹配：rising / peak / falling / resolution
- 留白技巧：Empty_Frame / Long_Hold / Off_Screen_Sound / Reflection_Shot

---

### 4. Suspense_Thriller_Designer — 悬疑惊悚镜头序列

**五大核心系统**：

**① Tension_Building（紧张感营造）**
- 三阶段：Setup(10-30s) → Escalation(20-60s) → Climax(5-15s)
- 探索场景标准序列：Wide_Entry → POV_Looking → Close_Inspection → Reaction_to_Find → Sound_Reaction → Dark_Corner_Focus
- 紧张技巧：Slow_Push_In / Sound_Design（心跳声、不规则音效）/ Lighting_Shifts / Isolation

**② Twist_Design（反转设计）**
- 四阶段结构：setup_phase → misdirection_phase → reveal_phase → aftermath_phase
- 反转类型：identity_reveal / situation_reveal / time_reveal
- 误导技巧：Visual_Distraction / Audio_Distraction / Expectation_Subversion / Character_Focus
- 隐藏线索技巧：Edge_Hiding / Depth_Hiding / Shadow_Hiding / Brief_Glimpse

**③ Jump_Scare（惊吓系统）**
- 时间轴：buildup(15-45s) → false_calm(2-5s) → trigger(0.1-0.3s) → reaction(1-2s) → aftermath(3-5s)
- 惊吓类型：Appearance_Scare / Sound_Scare / Cut_Scare / Mirror_Scare
- 效果因素：积累时间越长效果越强、声音设计占50%效果、每视频最多2-3次

**④ Suspense_Rhythm（悬念节奏）**
- 悬念曲线：build（逐渐上升）→ sustain（高位平台+虚惊）→ release（快速下降）
- pacing_rules：建立要慢、维持有波动、释放果断、释放后恢复

**⑤ Visual_Misdirection（视觉误导）**
- 隐藏技巧：Edge_Hiding / Depth_Hiding / Shadow_Hiding / Brief_Glimpse
- 揭示技巧：Pan_Reveal / Zoom_Reveal / Reflection_Reveal

---

### 5. Montage_Transition_Designer — 蒙太奇与转场序列

**五大核心系统**：

**① Time_Compression_Montage（时间压缩）**
- 训练蒙太奇结构：Beginning → Process（逐渐加快）→ Transformation（快剪）→ Climax（慢下来强调）
- 时间流逝指示器：natural（日出日落/月相/季节）/ artificial（时钟/日历）/ activity_based（重复活动）
- 节奏原则：开始慢建立起点 → 过程逐渐加快 → 蜕变快剪 → 高潮慢下来强调成果

**② Parallel_Narrative（平行叙事）**
- 四种结构：contrast_structure / convergence_structure / suspense_structure / thematic_structure
- 剪辑技巧：Rhythmic_Cutting / Match_Cut_Between_Lines / Sound_Bridge / Eyeline_Match
- 规则：节奏一致、使用相似动作/构图连接、音乐/声音延续

**③ Match_Transition（匹配转场）**
- 匹配类型：graphic_match（圆形/线条/形状）/ movement_match（转头/开门/伸手）/ color_match / sound_match / concept_match
- 经典模板：Eye_to_Sun / Door_to_Door / Hand_to_Hand / Object_to_Object / Reflection_to_Reality
- 实施步骤：识别共同元素 → 设计构图突出元素 → 确保位置大小相似 → 溶解或直接剪辑

**④ Transition_Effects（过渡效果库）**
- Cuts：Hard_Cut / Jump_Cut
- Dissolves：Cross_Dissolve / Fade_In_Out / Ripple_Dissolve
- Wipes：Edge_Wipe / Iris_Wipe / Clock_Wipe
- Others：Smash_Cut / L_Cut_J_Cut / Match_Cut

**⑤ Rhythm_Sync（节奏同步）**
- 同步类型：beat_sync / accent_sync / tempo_sync / emotional_sync
- 实施：分析音乐节拍 → 标记关键节拍点 → 安排镜头切换点 → 调整镜头时长匹配节奏


**垂直场景扩展**：
- 对话场景 → 竖屏：减少全景、增加面部特写、改变视线方向
- 动作场景 → 竖屏：动作方向改垂直、增加特写比例、节奏更快（平均镜头时长减少30-40%）
- 情感场景 → 竖屏：竖屏面部特写优势、减少水平移动全景

## 💡 核心理念

1. **分镜是AI视频的灵魂** — AI不知道什么是好的分镜，需要人来指导
2. **镜头序列有标准语法** — 好莱坞用近百年验证过的语法，AI视频同样适用
3. **场景分类是基础** — 先确定场景类型，再套用对应的镜头序列模板
4. **节奏决定观感** — 同样的镜头，不同的节奏，完全不同的效果
5. **实战驱动** — 所有Skill都设计为可直接用于AI生图/视频提示词
