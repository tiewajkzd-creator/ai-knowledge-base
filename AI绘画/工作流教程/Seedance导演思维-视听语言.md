## 一句话定位

同一套工具、同一个模型，有人出的是「AI 素材」，有人出的是「电影感片段」。差距不在提示词写得有多长，而在你脑子里有没有一点**影视基本功**。

---

## 一、视听语言基础

### 1.1 景别（镜头大小）

景别决定观众和角色的**心理距离**。

**《肖申克的救赎》结尾例子：**
- 先**远景** → 监狱外广阔世界（自由）
- 再**全景** → 人和暴雨环境
- 最后**近景/特写** → 面部表情，把「自由」变成可感知的情绪

**AI视频用法：** 悬崖对手戏：先用航拍大全景交代环境 → 再切近景/特写给表情（从容 vs 坚毅）。景别一递进，电影感就出来了。

---

### 1.2 镜头角度

角度直接影响观众对人物和场景的**主观感受**。

| 角度 | 效果 | 例子 |
|------|------|------|
| **仰拍** | 威严、危险 | 《低俗小说》马沙出场 |
| **俯拍** | 无力、渺小 | 《公民凯恩》童年小凯恩 |

**AI视频用法：**
- 治愈系骑行 → POV主观视角，代人入环境
- 朝堂/帝王 → 仰视强化威严

---

### 1.3 镜头运动

运动给画面**生命感和叙事张力**。

| 运动 | 效果 |
|------|------|
| **推镜头** | 从环境推向主体，强调人或物 |
| **拉镜头** | 从主体拉出，交代空间或收尾 |
| **希区柯克式变焦**（Dolly Zoom） | 机位后拉+镜头前推，背景变形、主体大小不变，产生眩晕不安 |

→ 提示词里写明「推/拉/摇/跟」等动词，让AI明确执行。

---

### 1.4 构图原则

| 构图 | 风格 |
|------|------|
| 中心对称 | 童话感、荒诞（《布达佩斯大饭店》） |
| 框架构图（门框/窗框） | 暗示被禁锢的命运（《末代皇帝》） |
| 三分法 | 通用平衡 |
| 主体居中 | 强调、仪式感 |

→ 提示词写明「对称构图/框架构图/三分法/主体居中」即可引导画面结构。

## 二、蒙太奇：镜头组接的学问

**蒙太奇 = 镜头的组接与排列，核心是 1+1>2。**

### 2.1 库里肖夫效应

同一个**面无表情**的镜头：
- 接一碗汤 → 觉得他在想"饿"
- 接棺材里的女人 → 觉得"悲伤"
- 接玩耍的小女孩 → 觉得"慈爱"

**结论：观众的情绪来自镜头和谁接在一起，而不是单镜头的表情本身。**

→ 做AI视频时，要特别注意**前后镜头的搭配**，而不只是打磨单镜。

### 2.2 交叉蒙太奇

《教父》洗礼段落：教堂里迈克尔在给侄子做洗礼，同时手下在各处执行暗杀。神圣与血腥、誓言与背叛并置。

**AI视频用法：** 双线叙事时，用提示词明确「A线/B线交替」「同一时刻不同空间」。

## 三、叙事结构

### 3.1 经典三幕结构

| 幕 | 占比 | 内容 |
|----|------|------|
| **第一幕** | ~25% | 建置——世界观、主角、日常、激励事件 |
| **第二幕** | ~50% | 对抗——冲突升级、试炼、中点转折、黑暗时刻 |
| **第三幕** | ~25% | 解决——最终对决、高潮、新平衡、主题升华 |

**关键节点：** 激励事件 → 中点逆转 → 一切尽失 → 高潮

**AI短片用法：** 先把「激励事件—中点—低谷—高潮」定下来，再给每个节点分配1～3个镜头。

---

### 3.2 英雄之旅

「英雄成长」模型，适合1～3分钟的完整小故事。

| 阶段 | 含义 |
|------|------|
| 已知世界 | 主角的日常 |
| 冒险召唤 | 打破日常的契机 |
| 跨越阈限 | 离开舒适区、进入未知 |
| 考验·盟友·敌人 | 途中遭遇与阵营 |
| 接近核心洞穴 | 逼近最大挑战 |
| 严峻考验 | 最低谷或决战 |
| 获得宝藏 | 得到力量/答案/物品 |
| 返回之路 | 回归途中再遇危机 |
| 复活重生 | 最终蜕变 |
| 回归并带来改变 | 回到日常但已不同 |

**AI视频用法：** 可截取「召唤→跨越→考验→严峻考验→归来」五段，每段1～2镜。

---

### 3.3 非线性叙事

打破时间顺序，能增加悬念和形式感。

**《记忆碎片》例子：** 彩色段落倒叙 + 黑白段落顺叙，两条线在中间汇合，形式与「失忆」主题统一。

**AI短片用法：**
1. 先定时间轴
2. 用视觉区分时间线（过去=暖黄/胶片感，现在=冷蓝/清晰）
3. 留好衔接点（重复元素）
4. 控制信息量（建议总时长1分钟内，时间线不超过2～3条）

## 四、声音设计

### 声音三要素

| 要素 | 作用 | 建议 |
|------|------|------|
| **对白** | 传递信息、塑造人物 | 有明确台词时保留；纯情绪片可弱化 |
| **音乐/BGM** | 定情绪基调、带节奏 | 情绪片：一条主旋律贯穿；节奏片：跟鼓点走 |
| **音效/SFX** | 增强真实感、强调动作 | 关键动作必加；环境声铺底；用静默制造反差 |

**分工原则：** 同一时刻不要三样都抢戏。

### 声音空间关系

| 概念 | 用法 |
|------|------|
| 画内/画外 | 画外脚步→下一镜再揭示，制造悬念 |
| 远近 | 远景用混响大、低频多的音效；特写用干净细节多的 |
| 混响 | 室内戏用短混响；旷野用长混响；水下用低通闷响 |

## 五、色彩理论

### 色彩心理效应

| 维度 | 冷暖 | 饱和度 | 色相 |
|------|------|--------|------|
| **暖色（橙红黄）** | 活力、怀旧、危险 | 高饱和：热烈、卡通感 | 红=激情/危险 |
| **冷色（蓝青紫）** | 冷静、疏离、科技、忧郁 | 低饱和：克制、写实、压抑 | 蓝=冷静/悲伤；绿=自然/诡异；黄=希望/不安 |

### 调色关键词对照

| 目标 | 关键词 |
|------|--------|
| 整体色调 | warm color grading / cold tone / teal and orange / muted colors |
| 色温 | golden hour / blue hour / daylight / tungsten |
| 电影感 | cinematic color grading / film grain / film look / Kodak / Fuji film |
| 风格化 | high contrast / soft contrast / bleach bypass |

**用法：** 在提示词末尾加一句调色描述，例如「warm orange and teal color grading, film grain, cinematic」。

## 六、导演风格与流派

### 重要电影流派

| 流派 | 视觉特点 | 提示词 |
|------|---------|--------|
| 黑色电影（Film Noir） | 高反差光影、百叶窗投影、烟雾、雨夜 | film noir / noir lighting / rain-soaked street |
| 赛博朋克 | 霓虹、雨、巨型建筑、高科技低生活 | cyberpunk / neon / rain / megacity / blade runner style |
| 新海诚 | 通透光影、天空与云、青春感 | Makoto Shinkai / anime / sky / clouds / lens flare / 光影通透 |
| 王家卫 | 霓虹、慢镜、大光圈虚化、抽帧 | Wong Kar-wai / neon / slow motion / shallow depth of field / 抽帧 |
| 香港武侠 | 竹林、水、衣袂、慢动作打斗、写意 | 武侠 / 竹林 / 慢动作 / 衣袂飘飘 / wuxia / wire fu |

### 当代重要导演风格

| 导演 | 视觉特点 | 提示词关键词 |
|------|---------|------------|
| 韦斯·安德森 | 对称构图、平面化、马卡龙色、横移跟拍 | symmetrical / centered / pastel / flat composition / horizontal tracking |

| 张艺谋 | 大色块、仪式感、中国符号（红、黄、水墨） | bold color blocks / red and gold / Chinese aesthetic / ritual |

## 七、剪辑节奏

### 剪辑基本原则

| 原则 | 说明 |
|------|------|
| **动作匹配** | 同一动作跨镜头衔接：前一镜抬手，下一镜接手部特写 |
| **视线匹配** | 上一镜某人看画外，下一镜给出他看的物体/人 |
| **180°轴线** | 对话/追逐时，机位保持在假想线同一侧，左右关系才不乱 |
| **动势衔接** | 用画面内运动方向接下一镜（出画/入画方向一致） |
| **节奏句** | 几个镜头为一组，形成「起—承—转—合」 |

### 节奏控制

| 类型 | 镜头长度 | 适用场景 |
|------|---------|---------|
| **快切** | 短镜头、高密度 | 紧张、追逐、冲突、音乐卡点 |
| **慢剪** | 长镜头、留白多 | 情绪、氛围、悬念 |
| **节奏变化** | 「慢—快—慢」或「快—更快—突然静」 | 制造对比和高潮 |

**AI短片节奏建议：**
- 开头1～2镜定调（环境/人物）
- 中段用3～5镜推进动作或情绪
- 结尾1～2镜收束或留白，避免突然断掉

## 八、AI时代的实际应用

### 用影视理论指导AI提示词

**公式：谁在做什么 + 景别 + 运镜 + 光影色彩 + 风格 + 画质**

**示例（中近景 + 推镜 + 王家卫感）：**
> A contemplative woman sitting by the window, soft afternoon sunlight streaming in, medium close-up shot, slow push-in camera movement, warm orange and teal color grading, cinematic, in the style of Wong Kar-wai, shallow depth of field, film grain.

### AI暂时替代不了的部分

**叙事逻辑：** 单段再好看，也要人来做顺序、转折、伏笔。
