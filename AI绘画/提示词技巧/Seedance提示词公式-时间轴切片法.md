## 核心方法论

### 八要素框架
```
主体 + 动作 + 场景 + 光影 + 镜头语言 + 风格 + 画质 + 约束
```

### 时间轴切片法（15秒视频标准结构）
- **0-3秒：** 全景 - 建立空间关系
- **3-7秒：** 中景 - 展现动作细节
- **7-11秒：** 特写 - 传递情绪张力
- **11-15秒：** 拉远 - 完整叙事收尾

## 实战案例

### 案例1：樱花树下的少年（日系青春）
```
9:16 竖屏，15 秒，日系青春电影质感。

0-4秒：全景俯拍，樱花花瓣纷纷扬扬从画面上方飘落，
日本神社古老的木质石板路铺满粉色花毯，镜头如呼吸般
缓慢下降穿过层层樱花。

4-8秒：中景跟拍，一位穿白色校服的年轻男生从远处走来，
黑色头发自然微卷，发丝随着步伐轻轻晃动，他低着头脚步
轻盈踩在落花瓣上发出细微的沙沙声。

8-12秒：近景侧拍，男生突然停下脚步抬起头，樱花花瓣从
镜头前快速划过形成虚影，他看向镜头方向，嘴角微微上扬
露出温柔的笑容，眼神清澈带着一丝腼腆，逆光中发丝边缘
泛起金色光晕。

12-15秒：特写侧脸，背景虚化成粉白色的樱花海洋形成唯美
散景，男孩的眼睛温柔地注视着镜头，镜头缓缓推进直到画面
占满整个面部，温暖治愈的氛围。

【强化词】4K 超高清，日系小清新色调，暖黄逆光效果，
景深感强，面部清晰稳定，五官全程一致不变形，人体比例
自然协调，动作流畅连贯无跳帧。
```

## 🧠 深度消化

### 1. 本质突破

**核心矛盾：** AI视频生成的"抽象描述 vs 具象控制"问题

- **传统方式：** "一个女生在海边跑步，日落很美"（太抽象）
- **时间轴切片法：** 按秒级拆解镜头，用导演语言描述

**突破点：** 把"感觉"翻译成"分镜脚本"

### 2. 可迁移的原则

**原则1：细节密度决定质量**
- 平庸："樱花飘落"
- 惊艳："樱花花瓣纷纷扬扬从画面上方飘落，踩在落花瓣上发出细微的沙沙声"

**原则2：强化词是约束，不是装饰**
- 作用：告诉AI"不要乱发挥"
- 例子："五官全程一致不变形"解决人脸崩坏

**原则3：镜头语言传递导演意图**
- 全景 = 空间关系
- 中景 = 动作展现
- 特写 = 情绪传递
- 推拉摇移 = 节奏控制

### 3. 与其他知识的关联

**关联：OpenClaw实践案例**
- 相同点：都强调"把抽象意图翻译成精确指令"
- OpenClaw：业务上下文 → 精确prompt → Agent执行
- Seedance：导演意图 → 分镜脚本 → AI生成

**启示：** AI工具的核心不是"更强的模型"，而是**更精确的输入**

### 4. 实践建议

**立即可用的模板：**
```

0-3秒：全景xxx。
3-7秒：中景xxx。

11-15秒：近景到全景xxx。
【强化词】关键词1，关键词2，关键词3。
```

## 作者经验总结

### 踩坑史
刚开始写提示词：
❌ "一个女生在海边跑步，日落很美"
→ 结果根本出不来惊艳的视频

研究上百个高质量提示词后发现：
✅ 差距不在模型，在提示词的细节密度

### 核心原则
> "写提示词从来没有高不可攀的门槛，唯一的秘密全在对细节的死磕里。空洞的词汇只能换来冰冷的像素，极致的微观描写才能赋予作品真正的灵魂。"

> "平庸和惊艳，也就是对光影和呼吸的毫厘刻画。"

### 实践建议
- Seedance 2.0 对提示词的遵循程度已经非常高
- 给出的指令越完善，得到的视频越给力
- 现在即梦的速度要快一些了
