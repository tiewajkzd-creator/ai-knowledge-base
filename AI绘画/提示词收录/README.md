# 🗂️ 提示词收录库（Prompt Vault）

**定位：** 收集、整理、复用已验证的优质提示词，作为创作养分长期沉淀。

---

## 收录范围

| 模型/平台 | 说明 |
|-----------|------|
| nanobanana | 重点收录 |
| Midjourney | MJ v6+ 提示词 |
| Stable Diffusion / SDXL | SD系提示词 |
| Flux | Flux系提示词 |
| ComfyUI 工作流 | 含工作流的提示词组合 |
| 其他 | Ideogram / Leonardo / 即梦等 |

## 收录格式（每条提示词）

```markdown
# [简短标题]

**目标模型：** nanobanana / MJ / SD / Flux / 其他
**来源：** [平台名称 - 作者/链接]
**发布时间：** YYYY-MM-DD
**质量评分：** ⭐⭐⭐⭐⭐
**成熟度：** 🌱 新鲜观点

## 提示词原文

（完整提示词）

## 参数/设置

- 比例：--ar 3:4
- 其他参数：...

## 效果说明

- 适用场景：人像/风景/概念艺术/...
- 风格标签：#日系 #柔光 #电影感
- 出图稳定性：高/中/低
- 备注：...

## 变体/衍生

- 变体1：将风格改为赛博朋克
- 变体2：...
```

## 使用方式

1. **收录**：看到好提示词 → 按格式存入本目录
2. **检索**：按模型/风格/场景搜索已收录提示词
3. **复用**：直接使用或基于已有提示词做变体
4. **进化**：记录每次使用效果，迭代优化提示词
