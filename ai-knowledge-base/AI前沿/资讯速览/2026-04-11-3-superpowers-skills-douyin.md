## 三个 Skills 简介

### 1. superpowers (4万星)
- **功能：** 模糊想法 → 发散思维 + 步骤分解 + 可执行方案
- **定位：** 创意激发 + 项目规划
- **GitHub：** github.com/模型社区/superpowers（待确认）

### 2. planning with files
- **功能：** 自动创建三个文档解决 AI 失忆问题，等于给 AI 配移动硬盘
- **效果：** 任务完成率提升 40%
- **定位：** 上下文管理 + 长期记忆

### 3. notebook LAM style
- **功能：** Claude Code 无缝对接 NotebookLM
  - 批量上传素材
  - 一键生成播客
  - 一键生成思维导图
  - 一键生成 PPT
  - 解析 YouTube 视频内容
- **定位：** 知识管理 + 研究助手

## 我的消化

**三个 Skills 的分工很清晰：**
- **superpowers** = 规划阶段（从模糊到具体）
- **planning with files** = 执行阶段（解决失忆、保持上下文）
- **notebook LAM style** = 收尾阶段（知识沉淀、成果输出）

这是一套完整的 AI 编程工作流：规划 → 执行 → 沉淀。

**与之前逐字稿内容的关联：**
- 上一个视频（@云桥网络）提到的 NotebookLM CLI 工具，和这里的 `notebook LAM style` 是同类工具，但前者是 CLI 版，后者是 Skill 版
- `planning with files` 这个概念在上一个视频里没有，是新增的重要工具

**评论区有用户提到：**
> "skills 限制了顶级模型的思维，其实这些 skills 也是模型写的，模型本来就懂"

这是一个值得思考的观点——Skills 的价值在于固化最佳实践，而非扩展模型能力本身。
