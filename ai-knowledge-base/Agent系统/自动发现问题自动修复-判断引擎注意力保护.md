# 多Agent系统自动发现问题、自动修复：判断引擎 + 注意力保护实战

## 核心思路

### 1. 判断力植入——自动发现问题

三种机制往 Issue Table 写问题：
- **Plugin Hook**：拦截异常
- **Cron Job**：定时扫描
- **Heartbeat**：心跳检查

三种来源同一个出口，Cron 定时扫表自动认领修复。

**效果**：Plugin v2 升级后每天条目从 6 条暴增到 150 条，90-95% 由系统自动发现+自动修复。

### 2. 注意力保护——不被打断

创建 **Feedback Skill**，把同步交互变成异步——提完 feedback 直接走人，Agent 下次心跳自动处理，不用实时响应。
