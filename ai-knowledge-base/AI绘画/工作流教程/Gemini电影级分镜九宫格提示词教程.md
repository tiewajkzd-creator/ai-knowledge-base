## 核心观点
AI 分镜的本质不是让模型帮你画画，而是你用文字当导筒，用提示词当场面调度。模型只是你的摄影师，你才是导演。

## 一、认知升级：Gemini 不吃标签，它吃故事

### 老一代 AI（标签式）
```
赛博朋克，雨夜，霓虹灯，男人，奔跑，8k，虚幻引擎渲染
```
→ 模型负责"拼凑元素"，元素之间的关系是乱的

### Gemini 时代（叙事式）
```
在一个赛博朋克风格的雨夜，一名身穿黑色风衣的男子在霓虹灯闪烁的湿滑街道上狂奔。
雨水在前景中形成了模糊的线条，背景是高耸入云的钢铁建筑。
```
→ 模型能精确分配每个元素的图层关系——谁在前景、谁在背景、谁在动、谁是静的

**核心原则：用叙事代替标签。**

## 二、四大支柱（写对四件事，出图不会太差）

| 支柱 | 问题 | 例子 |
|------|------|------|
| **1. 你是谁**（角色设定） | 告诉 Gemini 它的身份 | "你是一位拥有20年经验的好莱坞分镜师" |
| **2. 做什么**（具体任务） | 明确输出规格 | "生成一张16:9的电影级分镜，使用中景构图" |
| **3. 背景是什么**（上下文） | 这是画面的"Why" | "这是主角刚经历重大失败后的深夜" → 光线自动压暗、色调偏冷 |
| **4. 交付格式**（物理规格） | 画幅/文字/排版 | "16:9画幅，无文字" |

## 三、每张分镜图的万能公式

```
主体 + 动作 + 环境 + 镜头语言 + 风格氛围
```

**示例：**
```
一位25岁的亚洲青年男性（主体），
正站在便利店橱窗前凝视窗内（动作），
深夜的雨后街道，地面反射着荧光灯的暖色（环境），
中景平视构图，半身被暖光照亮、半身被冷青色阴影吞没（镜头语言），
电影级 Teal-and-Orange 调色，Kodak Vision3 500T 胶片颗粒感（风格氛围）。
```

### 三个细节要求
1. **主体要具象**：不是"一个男人"，是"25岁、短发微乱、穿深色连帽卫衣、背旧帆布包的亚洲青年"
2. **动作要用强动词**：不是"在街上"，是"正倾身狂奔"或"双手插兜低头行走"
3. **情感别写抽象的**："悲伤"不行，要写"双肩下垂，目光呆滞地盯着地面，脸上有未干的泪痕"

## 四、镜头语言速查表

### 景别速查表
| 怎么说 | 画面效果 | 什么时候用 |
|--------|----------|------------|
| Extreme Long Shot（极远景） | 人在画面里小到几乎看不见 | 开篇建立世界观 |
| Wide Shot（远景） | 全身入画，环境清晰 | 交代人物与环境的关系 |
| Medium Shot（中景） | 腰部以上 | 日常对话、从容叙事 |
| Medium Close-Up（中近景） | 胸部以上 | 表情开始成为重点 |
| Close-Up（特写） | 整个头部 | 情感爆发，强调面部细节 |
| Extreme Close-Up（极大特写） | 只有眼睛或嘴唇 | 心理压迫，关键线索 |

### 角度速查表
| 怎么说 | 心理效果 |
|--------|----------|
| Eye Level（平视） | 平等、客观 |
| Low Angle（仰拍） | 主体高大、有威胁感 |
| High Angle（俯拍） | 主体渺小、脆弱 |
| Dutch Angle（荷兰角） | 不安、混乱、心理失衡 |

### 光线速查表
| 怎么说 | 视觉效果 |
|--------|----------|
| Three-point softbox（三点柔光） | 干净均匀，商业人像感 |
| Chiaroscuro / Rembrandt lighting（伦勃朗光） | 高对比，一半亮一半暗，戏剧感拉满 |
| Golden hour backlighting（黄金时刻逆光） | 温暖怀旧，轮廓光 |
| 85mm portrait lens, f/2.0 | 人像黄金焦段，完美虚化背景 |
| Kodak Vision3 500T | 电影级胶片色彩科学 |

**原则：不要写"好看的光线"，要写"伦勃朗光，打火机火焰在左脸形成暖橙三角"。越具体，AI越老实。**

## 五、角色一致性：分镜的终极痛点

**解决方案："锚定与枢轴"三步走**

### 第一步：生成角色基准图
用以下提示词生成一张满意的"基准脸"，这张图就是整套九宫格的"人物身份证"：

```
一位约25岁的亚洲青年男性的正面半身肖像照。
短发微乱、被雨水打湿，身材精瘦，下颌线条分明，肤色自然偏暖。
穿深灰色棉质连帽卫衣（帽子放下），黑色直筒裤，
斜挎一个旧橄榄绿帆布包，包上有一个黄铜搭扣。
表情是安静的疲惫与内省，目光直视镜头。

拍摄方式：85mm人像定焦镜头，f/2.0光圈，
柔和的影棚灯光搭配微弱的冷青色辅助光，
干净的深灰色背景，浅景深，
面部细节极致清晰——皮肤纹理、毛孔、眉毛走向均可见。
电影级质感，Kodak Vision3 500T胶片颗粒感。
1:1画幅比例。
```

> 关键：背景要干净、光线要均匀、正面平视、表情中性。越朴素越稳定。

### 第二步：上传为参考图
把基准图上传到 Gemini，勾选"作为参考"。Gemini 会把特征编码成数学约束，"锁死"后续每帧的脸。

### 第三步：枢轴扭转
后续提示词只需描述新场景和新动作，同时加一句保险：
> "指令红线：绝对保持面部解剖结构、发型与固有肤色特征不变。"

**如果连续生成几十帧后脸开始崩**：换背影、过肩镜头、全景、剪影。用电影语言掩盖技术短板，效果反而更高级。

## 六、实战：夜晚街头人物九宫格「漫游者/NIGHT WALKER」

### 主题设定
一位25岁的亚洲青年，深夜独自行走在霓虹闪烁的雨后街头。情感弧线从迷茫压抑，经过沉思驻足，最终走向微光中的释然。


### 情感弧线设计
```
帧1 城市全景 → 冷漠、庞大
帧2 霓虹特写 → 疏离、赛博感
帧3 雨巷背影 → 孤独、迷失
帧4 侧脸点烟 → 沉思、挣扎
帧5 积水倒影 → 自我碎裂
帧6 便利店前 → 短暂温暖      ← 情绪开始转折
帧7 天桥俯拍 → 渺小感
帧8 回眸微笑 → 释然、接纳     ← 核心转折点
帧9 路灯剪影 → 余韵、继续前行
```

### 统一风格后缀（每帧都要加）
```
视觉风格：电影级新黑色风格，带赛博朋克霓虹美学。
模拟 Kodak Vision3 500T 钨丝灯平衡电影胶片，带明显有机颗粒感。
Teal-and-Orange 互补色调色，深黑色压低，
中间调去饱和但霓虹重点色选择性穿透。
浅景深配奶油般的六边形散景光斑。
雨水放大所有表面的反射。
沉郁、大气、深度城市感、情感共鸣。16:9 画幅比例。
```

## 七、可直接替换的变体方向

| 主题 | 情感弧线 | 色彩体系 | 关键场景替换 |
|------|----------|----------|-------------|
| 雨天咖啡馆 | 等待→邂逅→离别 | 暖琥珀+雾灰 | 窗边座位、拿铁拉花特写、雨伞 |
| 深夜末班车 | 疲惫→恍惚→到站 | 荧光白+深蓝 | 空车厢、车窗倒影、到站指示灯 |
| 凌晨天台 | 压抑→呐喊→黎明 | 深紫+破晓橙 | 城市夜景、风吹发丝、第一缕阳光 |
| 老街漫步 | 怀旧→发现→告别 | 褪色暖黄+青绿 | 老招牌、石板路、转角回眸 |

## 八、完整工作流

1. **定角色**：写好角色锚定文本，生成一张满意的"基准脸"
2. **定风格**：确定风格后缀，锁死色彩体系和质感
3. **拆场景**：按情感弧线把故事拆成9个视觉节拍
4. **配镜头**：每个场景分配景别+角度+光线类型
5. **逐帧生成**：角色锚定+场景描述+风格后缀，一帧一帧生成
6. **拼贴成图**：用Canva/Figma/PPT拼成3x3九宫格
